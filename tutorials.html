New Klusterbox Version 3.0 Tutorials Are Coming Soon

I recorded and posted some tutorials on youTube. These tutorials should give you a quick overview and explain how to use Klusterbox. I'm going to be working on a text document which will give written step by step instructions, but that will probably take a while to finish. So in the meantime, be sure to watch the videos if you have any questions.

All of these video were made with Klusterbox version 1.0. So the user interface might look different in parts. That doesn't matter though. The functionality will be the same. The automated features are not covered. Stay tuned for those tutorials.

What is Klusterbox?

This provides a brief over view and should give you a good idea of the possiblities of the program. This video covers version one in the first part - this is the original video promoting Klusterbox, then covers version two and three additional functionality after time 3:39.

Download and Install

This video will give you an idea of how to download and install. Klusterbox does not need to setup on your computer, it already runs without needing to set up. Just download it, move it to where you want it and open it. You might want to make a shortcut for your desktop. The tutorial covers that too. The place where you go to download Klusterbox has changed since this video was made. Now you go to the Downloads link at the top of this page.

Building and Maintaining Carrier List

Learn how to build carrier list. This video will demonstrate how to enter in new carriers. All carriers have attributes like name, ns day, list status, route and station. Manage these attributes day to day or week to week. Learn how these the carrier list is time and station sensitive.

Inputting Clock Rings

Now you're getting into the main work of conducting the improper mandate investigations. If you don't have the appropiate digital files, you will want to know how to use Klusterbox to enter clock rings into the database. The video explains the 5200, RS and moves fields. Learn about multiple move functionality - a which tracks multiple moves off a carrier's route on a given day.

Generating Spreadsheets

Now you are ready for the payoff. Klusterbox can generate a report and a spreadsheet. The spreadsheet opens in Excel or other compatable spreadsheet programs like Libre Calc. Just print off the spreadsheet for your informal A meeting and you'll be ready to make your case. Learn about options for spreadsheet readers, tolerances and altering the spreadsheet after it's been generated. This tutorial shows you how.

PDF Converter

So you've submitted an information request asking for the Employee Everything Report in both csv and pdf format, but what you've gotten was the Employee Everything Report in just the pdf format. Since Klusterbox needs the csv format, you need a way to convert that PDF into a csv file which Klusterbox can read. Well, the PDF Converter can convert that pdf into a csv file which Klusterbox can read.
