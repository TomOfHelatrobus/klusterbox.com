Klusterbox Installer for Version 4.001 is Here!

What is Klusterbox?
  ● Klusterbox is a tool for investigating article 8 grievances and was developed by a NALC union steward.
  ● Klusterbox is a computer program available for Windows and Mac OS.
  ● Klusterbox is free to download and use.
  ● Development is ongoing. More features and functionality are on the way.

Klusterbox Features:
  ● Build list of carriers. Revise, update and change the list from week to week as needed. Klusterbox keeps track of who's on the OTDL, ns/days, who's on which route and what station they're at.
  ● Data entry screens allow for easy entry of clock rings. Klusterbox comes with multiple move functionality - record whenever carriers move off their routes, even when when they do it multiple times in a day.
  ● Don't know what color ns/day correlates to which day of the week? No problem, Klusterbox matches the color ns/day code to the correct day on in any given service week.
  ● Choose between investigating just one day or an entire service week with the investigation range feature.
  ● Klusterbox can be used at the branch level to conduct investigations for multiple stations
  ● Use codes to keep track of notable cases like when OTDL carriers call in sick or when non-list carriers are mandated to work on their ns/days.
  ● View the results of the investigation in reports or generate a spreadsheet which you can print. The spreadsheets contain all the necessary formulas allowing you to update them outside of Klusterbox.
  ● Utilize tolerances to cut down on the clutter and ignore negligible violations. Adjust the tolerances to suit your preferances or turn them off altogether.
  ● Automated features such as Automatic Data Entry allow you to speed up the process. Do a two hour investigation in less than five minutes.
  ● Find violations of the 12/60 hour rule with the Automatic Over Max Finder and generate a report with just a few mouse clicks.
  ● Generate a spreadsheet for violations of the 12/60 hour rule the Over Max Spreadsheet. The spreadsheet takes into account paid leave, whether or not a carrier is on the OTDL and stops counting daily violations after 60 hours in a week. Another sheet will provide you with a simple list of carriers and violations.
  ● Generate a report for pay periods for any year you want with the Pay Period Guide Generator. The report gives you the pay period, start and end date. Klusterbox stamps your spreadsheets and reports with the correct pay period. Don't be inconvenienced again.
  ● Keep track of which settlements have been paid and which haven't been paid wirh Informal C, a tool which tracks what management owes, down the the individual carrier.
  ● Past versions of Klusterbox depended upon the employee everything report in the csv format, but the new PDF Converter allows access to the suite of Klusterbox functionality with just an employee everything report in pdf format. This makes your information request easier and less complicated.
  ● Version 4.0 has an installer for Windows. Now just download and install Klusterbox with the installer. Klusterbox will show up in your start menu like a normal program.
  ● Use Database Maintenance tools to see what you have in the database. You can delete all records for carriers who you no longer want in the system and for all other records you can view, delete or even reset the entire database and start from scratch.
  ● Starting in version 4.001, Klusterbox is now compatable with five digit route numbers for route numbers over 99 and PTFs are now recognized.

In the most recent version, I've expanded Klusterbox to allow route numbers over 99 and ptf are allowed as a list type. Moreover, these changes are reflected in the Automatic Data Entry. PTFs will appear in the Improper Mandates spreadsheet as Auxiliary carriers. Also, I've added Database Maintenance tools to help you keep your database clean and to help you know what is in it.

In version 4.0, finally Klusterbox has an installer. You can start up Klusterbox from the start menu and you no longer have to keep track of the kb_sub folder which was an issue in version 3 and earlier. It's like a REAL program now. If you don't like it, Klusterbox has an uninstaller - just uninstall it like a regular program.

In version 3.008, I've developed a Mac compatable version of klusterbox. This works like a proper app which is placed in the Applications Folder can be accessed from Launch Pad (that's an upgrade from the Windows version). This was more work that I was expecting and I had to make a good deal of modifications but I did learn a lot.

In version 3.007, I'm working to improve the pdf converter which converters employee everything reports in pdf format to csv format. Klusterbox can read csv formatted employee everything reports and allow you to skip the hours needed to enter in those carrier rings. Extracting data from pdf files is like pulling teeth, so there have been a lot of bugs along the way, but I think this is the best version so far and I'm hoping I've solved the problem.
I'm starting on a series of reports which open in notepad. So far I've got carrier routes and ns days, but future versions will have a lot more such as OTDL equitability and distribution of overtime for carriers not on the OTDL.
